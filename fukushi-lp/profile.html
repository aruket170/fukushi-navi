福祉サポートナビ 運営

障害福祉の制度は分かりにくく、初めて相談する方にとって情報が散在しがちです。 当サイトは、障害福祉サービスや手当の受給要件をやさしく整理する参考ツールとして、 無料で公開しています。行政の公式判断に代わるものではありません。

サイトの目的

  • ステップ形式の質問により、受給の可能性がある制度を一覧で把握できるようにする
  • お住まいの市区町村名に基づき、登録事業所をカード形式で表示する
  • 公的窓口への相談を促し、情報格差の緩和に資する

運営方針

  • 掲載内容は可能な限り一次情報（厚生労働省・自治体の案内等）に基づき更新します
  • 個別の医療・福祉判断は行いません
  • 広告（Google AdSense 等）を掲載する場合も、コンテンツの信頼性を損なわないよう配慮します

技術・ホスティング

静的サイト（HTML / CSS / JavaScript）を GitHub で管理し、Cloudflare Pages で配信しています。 事業所データは Supabase の PostgreSQL データベースから読み取り専用で取得しています。
